# Zerro AI VNext Project

### 🎯 사령관 지시사항
> 테트리스 게임 만들어봐

### 🏗️ Architecture
* 게임 시작
	+ 게임 보드 생성
	+ 테트리스 블록 생성
* 게임 진행
	+ 사용자 입력 처리
		- 왼쪽 키: 블록 이동
		- 오른쪽 키: 블록 이동
		- 아래쪽 키: 블록 이동
		- 위쪽 키: 블록 회전
	+ 블록 충돌 처리
		- 보드 하단 충돌
		- 다른 블록 충돌
	+ 블록 고정
	+ 한 줄 완성 처리
		- 해당 줄 삭제
		- 점수 증가
	+ 레벨 증가 처리
		- 블록 떨어지는 속도 증가
* 게임 종료
	+ 게임 오버 상황 판단
	+ 사용자의 최종 점수 표시
	+ 게임 재시작 또는 종료 선택
