# Zerro AI 5-Stage Project

### 🎯 사령관 지시사항
> 테트리스 게임 만들어줘

### 🏗️ Architecture
- 게임 시작
  - 게임 보드 생성
    - 10x20 격자 생성
  - 테트로미노 생성
    - 7가지 종류(I, J, L, O, S, T, Z)
    - 각 테트로미노 4개의 정사각형 블록으로 구성
- 플레이어 조작
  - 테트로미노 이동
    - 왼쪽, 오른쪽, 아래쪽 이동
  - 테트로미노 회전
- 테트로미노 놓기
  - 이미 놓여진 블록과 겹치지 않도록
  - 게임 보드의 경계를 넘어가지 않도록
- 줄 삭제
  - 한 줄 완전히 채워질 때마다
  - 해당 줄 삭제
  - 플레이어 점수 획득
- 게임 종료
  - 게임 보드의 위쪽에 블록 쌓여서 새로운 테트로미노 놓을 수 없을 때
  - 플레이어 점수 표시
