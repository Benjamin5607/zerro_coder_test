# Zerro AI VNext Project

### 🎯 사령관 지시사항
> 테트리스 게임 하나 만들어봐

### 🏗️ Architecture
- 게임 시작
  - 게임 보드 생성
    - 크기: 10x20
  - 테트로미노 생성
    - 종류: 7가지 (이, 지, 테, 오, 에스, 제이, 자이)
    - 생성 방식: 무작위
- 사용자 입력
  - 이동
    - 왼쪽
    - 오른쪽
    - 아래쪽
  - 회전
    - 위쪽
- 테트로미노 고정
  - 조건
    - 게임 보드의 아래쪽에 도달
    - 다른 테트로미노와 충돌
- 게임 보드 업데이트
  - 한 줄 완성
    - 해당 줄 삭제
    - 점수 증가
  - 게임 난이도 증가
- 게임 종료
  - 조건
    - 게임 보드 꽉 찼을 때
  - 사용자 점수 저장
  - 최고 점수 표시
